# 🏠 Calculadora de Despesas de Imóveis

Aplicação web feita com Streamlit para simular os custos envolvidos na compra de imóveis financiados em Goiânia, Trindade, Senador Canedo e Aparecida de Goiânia.

## 💻 Acesse

Hospede este app facilmente no [Streamlit Cloud](https://streamlit.io/cloud)

## 📦 Requisitos

- Python 3.8+
- Streamlit

## 🚀 Como usar localmente

```bash
pip install -r requirements.txt
streamlit run app.py
